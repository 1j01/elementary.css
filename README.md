
# elementary.css

elementary.css is a
[conversion](http://github.com/1j01/postcss-gtk.git) of
elementary OS's
[GTK stylesheet](https://launchpad.net/egtk)
to browser CSS.

[**Check out the demo!**](http://1j01.github.io/elementary.css/demo/)

[elementary OS](https://elementary.io/) is pretty.
